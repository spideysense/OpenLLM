# Skill: Marketing & Creative

## Ad Copy
### Headlines (max 30 chars for Google, 40 for Facebook)
- Lead with benefit, not feature
- Use numbers: "Save 3 hours/day" not "Save time"
- Create urgency: "Limited", "Today only", "Before it's gone"
- Ask questions: "Still paying for AI?"

### Body Copy
- AIDA: Attention → Interest → Desire → Action
- PAS: Problem → Agitate → Solve
- One CTA per ad. Make it specific: "Start free trial" not "Learn more"

## Landing Pages
Structure (top to bottom):
1. **Hero**: Headline + subhead + CTA + visual (above the fold)
2. **Social proof**: logos, testimonials, numbers ("10,000+ users")
3. **Problem**: describe the pain vividly
4. **Solution**: how your product solves it (3 key features)
5. **How it works**: 3-step process with icons
6. **Testimonials**: real names, photos, specific outcomes
7. **Pricing**: clear, simple, highlight recommended plan
8. **FAQ**: address top 5 objections
9. **Final CTA**: repeat the hero CTA

## Email Marketing
- Subject line: 6-10 words, create curiosity or urgency
- Preview text: complement the subject, don't repeat it
- Body: one topic, one CTA, under 200 words
- P.S. line: second chance to convert (most-read part after subject)
- Send time: Tuesday-Thursday, 10am local time

## Social Media
### Twitter/X
- Under 200 chars performs best (even though limit is 280)
- Thread format for long content (1/🧵)
- Hook in first tweet, value in middle, CTA at end
- Quote tweet > reply for engagement

### LinkedIn
- First line is the hook (shows before "see more")
- Use line breaks for readability
- Personal story + lesson format performs best
- 1,300-2,000 chars optimal
- Post 8-10am weekdays

### TikTok/Reels
- Hook in first 3 seconds
- Text overlay for silent viewing
- Trending audio when relevant
- CTA: "Follow for more" or "Link in bio"

## SEO Content
- Target one primary keyword per page
- Keyword in: title, H1, first paragraph, URL slug, meta description
- 1,500-2,500 words for pillar content
- Include related keywords naturally (LSI)
- Internal link to 2-3 related pages
- External link to 1-2 authoritative sources
- Add Schema.org markup (Article, FAQ, HowTo)

## Creative Assets (HTML-based)
When asked for banners, social graphics, or ads:
- Create as HTML/CSS with exact pixel dimensions
- Use CSS grid/flexbox for layout
- Export-ready: user can screenshot or use html2canvas
- Common sizes: 1200×628 (OG/social), 1080×1080 (Instagram), 1080×1920 (Stories)
