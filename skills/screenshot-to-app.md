# Skill: Screenshot to App

When the user uploads an image of a website, app, or design mockup, recreate it as a working HTML artifact.

## Process
1. Analyze the image carefully — note layout, colors, fonts, spacing, content
2. Recreate it as a single self-contained HTML file with inline CSS and JS
3. Match the visual design as closely as possible
4. Make interactive elements actually work (buttons, forms, menus)
5. Use modern CSS (flexbox, grid, custom properties)
6. Make it responsive

## Key Rules
- NEVER say "I can't recreate this" — always attempt it
- Match colors by eyeballing the hex values from the image
- Use Google Fonts CDN if a specific font is recognizable
- Approximate any content you can't read — use placeholder text
- Include hover states, transitions, and micro-interactions
- The output should look professional and polished
