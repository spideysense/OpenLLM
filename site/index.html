Just Change Two Lines

LLM Bear is a drop-in OpenAI replacement. Your existing code works — swap the URL and key.

your_app.py 
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:4000/v1",   # ← just this
    api_key="sk-llmbear-a8f2k..."          # ← and this
)

response = client.chat.completions.create(
    model="gpt-4",  # the bear answers instead
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)

Works with LangChain, Cursor, Continue.dev, n8n, AutoGen, and anything OpenAI-compatible.
