# Screenshots

Drop PNGs here and they appear automatically on the homepage gallery and in the
repo README. Expected filenames (capture at a clean, high-res window size):

- `chat.png`     — the desktop chat answering something real
- `tools.png`    — a turn where it calls web search / runs a command (show the tool step)
- `voice.png`    — voice mode full screen (the orb + transcript)
- `ios.png`      — the iPhone app in a conversation
- `artifact.png` — a rendered artifact (code or an HTML/SVG preview)

The homepage only shows a tile once its image loads, so partial sets look fine.
