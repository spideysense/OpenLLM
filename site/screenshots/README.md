# Screenshots

Drop PNGs here and they appear automatically on the homepage gallery and in the
repo README. Expected filenames (capture at a clean, high-res window size):

- `chat.png`      — the desktop chat answering something real
- `voice.png`     — iPhone voice mode full screen (the orb + "Listening…")
- `tools.png`     — the Tools settings list (web search, calculator, run commands, git…)
- `vision.png`    — iPhone reading an uploaded photo with a local vision model
- `artifact.png`  — a rendered artifact (code or an HTML/SVG preview)
- `websearch.png` — iPhone answering from a live web search, with the source cited
- `models.png`    — the model library / picker
- `memory.png`    — the World Model (local memory) settings
- `webapp.png`    — the browser app reaching your own machine
- `savings.png`   — the home dashboard "saved vs cloud" hero

Crop browser/OS chrome and anything machine-specific (tunnel URLs, API keys,
real key labels) before adding a shot here — this folder ships to the public site.

The homepage only shows a tile once its image loads, so partial sets look fine.
